# Dots

## This has not been updated in a while.

These the my dotfiles for my first Arch Linux rice. Inspired by [this post](https://www.reddit.com/r/unixporn/comments/mxcru3/bspwm_learning_openframeworks/). 

![Screenshot](/.screen.png)

## Details
* **OS** - Arch
* **WM** - bspwm
* **Bar** - polybar
* **Application Launcher** - rofi
* **Web Browser** - firefox with flyfox
* **Text Editor** - NeoVim
* **Terminal** - urxvt
* **File Manager** - Ranger
* **PDF Reader** - Zathura
* **Font** - Tamzen
